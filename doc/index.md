# mdev 开发文档

### 编码规范
* 所有文件名、文件夹名使用“-”链接单词。
* 所有文本文件使用utf-8格式。
* 所有文本文件使用linux换行符。
* 所有文本文件使用4个空格，代替tab键值。

### 项目结构
src为源代码目录，通过执行mdev build生成build输出目录。
```
src
|-- conf
|-- layout
|-- module
|-- page
|-- ...
build
|-- page
|-- resource
```
